# FinSmart

FinSmart is a personal finance manager that helps you track
your income and expenses, set weekly and monthly budgets,
and make smarter financial decisions powered by Machine Learning.

---

## Features
- Track income and expenses across multiple accounts
- Set weekly and monthly budgets by category
- ML-powered spending predictions
- Unusual spending detection
- AI-driven saving suggestions

---

## Tech Stack
**Backend:** Python, FastAPI, PostgreSQL, SQLAlchemy  
**Frontend:** React, Chart.js, CSS  
**Machine Learning:** scikit-learn, pandas

---

## Requirements
- Python 3.13+
- Node.js 24+
- PostgreSQL
